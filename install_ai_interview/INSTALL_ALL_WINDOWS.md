# AI 面试官 - 极简部署指南 (Windows)

欢迎使用 AI 面试官！
本压缩包内置了全自动环境配置脚本，它会**自动帮你从云端下载项目代码本体**、安装 Git、Miniconda、创建 Python 环境并安装所有依赖。

## 1. 运行一键安装脚本

请确保你已经将本压缩包解压到了电脑上的某个文件夹中。
直接在解压后的文件夹内，双击运行 `install.bat` 即可开始自动配置！

*(注意：如果双击后一闪而过或者提示权限错误，请右键点击 `install.bat` 并选择“以管理员身份运行”)*

> 脚本运行期间会自动打开命令行黑框，请耐心等待其下载和安装，中间可能需要几分钟的时间。

*(执行完毕后，当前目录下会自动生成一个名为 `ai_interview` 的文件夹，这就是真正的项目源码！)*

---

## 2. 填写 API 密钥

安装成功后，进入刚刚生成的 `ai_interview` 项目目录，你会发现里面自动生成了一个 `.env` 隐藏文件（如果你看不到它，请在文件夹顶部的“查看”菜单中勾选“隐藏的项目”）。

请用记事本打开它。你需要前往火山引擎申请豆包大模型的 API Key，**具体申请步骤请参考本文件夹内的《豆包_API配置方式.docx》文件**。申请完毕后将其填入 `.env` 中：

```env
DOUBAO_API_KEY=你的API_KEY（仅填写本项即可，其余无需修改）
```

---

## 3. 启动并访问

一切就绪后，你需要通过命令行来启动服务。

按 `Win + R` 并输入 `cmd` 打开命令行工具（**注意：以后每次你想启动服务时，都必须先使用 `cd` 命令进入 `ai_interview` 这个项目文件夹！**）：

> **提示**：如果不知道怎么写路径，可以在命令行里先敲一个 `cd `（注意 `cd` 后面有一个空格），然后**把刚才生成的 `ai_interview` 文件夹直接拖拽进黑框里**，按下回车键即可进入目录。

进入目录后，执行以下两条命令启动：
```bash
# 激活虚拟环境
conda activate ai_interview

# 启动服务
python -m uvicorn app:app --reload
```

看到类似 `Application startup complete.` 的提示后，打开浏览器访问 [http://127.0.0.1:8000](http://127.0.0.1:8000) 即可开始体验！

> **注意**：如果一键脚本运行失败（如网络原因导致下载中断），请参考项目内的 `docs/INSTALL.md` 逐步排查并安装。
