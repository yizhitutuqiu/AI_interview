# AI 面试官 - 极简部署指南 (macOS / Linux)

欢迎使用 AI 面试官！
本压缩包内置了全自动环境配置脚本，它会**自动帮你从云端下载项目代码本体**、安装 Git、Miniconda、创建 Python 环境并安装所有依赖。

## 1. 运行一键安装脚本

请确保你已经将本压缩包解压到了电脑上的某个文件夹中。
打开你的终端（Terminal），使用 `cd` 命令进入本压缩包解压后的目录（也就是 `install.sh` 所在的目录）。

> **提示**：如果不知道怎么写路径，可以在终端里先敲一个 `cd `（注意 `cd` 后面有一个空格），然后**把解压后的整个文件夹直接拖拽进终端窗口里**，按下回车键即可。

进入目录后，执行以下两条命令：
```bash
chmod +x install.sh
./install.sh
```

*(执行完毕后，当前目录下会自动生成一个名为 `ai_interview` 的文件夹，这就是真正的项目源码！)*

---

## 2. 填写 API 密钥

安装成功后，进入刚刚生成的 `ai_interview` 项目目录，你会发现里面自动生成了一个 `.env` 隐藏文件（部分系统可能需要使用快捷键 `Command + Shift + .` 开启“显示隐藏文件”）。

请用文本编辑应用（如 TextEdit、VS Code）打开它。你需要前往火山引擎申请豆包大模型的 API Key，**具体申请步骤请参考本文件夹内的《豆包_API配置方式.docx》文件**。申请完毕后将其填入 `.env` 中：

```env
DOUBAO_API_KEY=你的API_KEY（仅填写本项即可，其余无需修改）
```

---

## 3. 启动并访问

一切就绪后，在命令行中输入以下命令启动服务（**注意：以后每次你想启动服务时，都必须先打开终端，使用 `cd` 命令进入 `ai_interview` 这个项目文件夹！**）：

```bash
# 1. 确保你当前在 ai_interview 目录下
cd ai_interview

# 2. 激活虚拟环境并启动
conda activate ai_interview
python -m uvicorn app:app --reload
```

随后打开浏览器访问 [http://127.0.0.1:8000](http://127.0.0.1:8000) 即可开始体验！

> **注意**：如果一键脚本运行失败（如网络原因导致下载中断），请参考项目内的 `docs/INSTALL.md` 逐步排查并安装。
